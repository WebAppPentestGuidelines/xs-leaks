# 安全なデフォルト

このセクションには、2種類の安全なデフォルトについて説明している記事を掲載しています。

* [Partitioned Caches]({{< ref "./partitioned-cache.md" >}}) – キャッシュリソースを異なるサイト間で共有できないようにします。
* [Cross Origin Read Blocking (CORB)]({{< ref "./corb.md" >}}) – 特定の種類のレスポンスが特定のクラスのリクエストによって参照されるのを防ぎます。
